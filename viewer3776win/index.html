Highlights

The design prioritizes viewer comfort while keeping broad format support inside a single portable executable.

Spread view and seek preview for comics Built to switch previews quickly while you scrub through pages, including spread layouts.
Handles more than images Open PDFs, text, HTML, RTF, video, and audio with the same viewer flow.
Portable operation Runs without installation and keeps settings plus history inside the `_res` folder.

Supported Formats

Formats are grouped by what works out of the box and what expands with optional helpers.

Images
JPG, PNG, WebP, AVIF, HEIC, SVG, GIF, BMP
Documents
PDF, TXT, LOG, INI, XML, Markdown, HTML, RTF
Media
MP4, MOV, WMV, AVI, MKV, WebM, MP3, WAV, M4A, FLAC, WMA
Archives
ZIP / CBZ work by default. RAR / 7z / LZH / LHA require 7z.dll.
ffmpeg.exe support
Older FLV files can be auto-converted for playback when ffmpeg.exe is available.

Archive support RAR / 7z support is not bundled. Place 7z.dll in `_res\bin` when you need those formats.
Video playback Older containers and codecs still depend on OS media support, with ffmpeg.exe used as an optional helper when needed.
